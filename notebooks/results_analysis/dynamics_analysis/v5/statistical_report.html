5. Summary of Significant and Marginal Effects

H condition (no AI)

No significant effects found on any DV for either target task. The only marginal result is cognitive load on the Low/Low target when compared to High/Low starters (d = +0.60, pperm = .054). Task order has no detectable effect without AI assistance.

H+AI condition — Significant results

  • Trust on Low/Low target — Participants who did Low/Low first reported lower trust (M = 2.15) than those who started with any other task. Strongest contrast: vs High/Strong starters (d = −0.94, pperm = .008). Binary: d = −0.71, pperm = .007. KW omnibus marginal (p = .050).
  • Reliance on High/Strong target — KW omnibus significant (H = 12.77, p = .005, η² = .118). Target-first group shows higher reliance than Low/Low starters (d = +0.69, pperm = .038).
  • Over-reliance on High/Strong target — KW significant (H = 15.04, p = .002, η² = .145). Complex pattern: Low/Strong starters show very high over-reliance (M = 0.98), marginally higher than target-first group (d = −0.64, pperm = .053).
  • Under-reliance on High/Strong target — KW significant (H = 9.42, p = .024, η² = .077). Low/Low and High/Low starters show marginally higher under-reliance (both pperm = .080).
  • Cognitive Load on High/Strong target — KW significant (H = 8.23, p = .042). High/Low starters report lower load (d = +0.81, pperm = .014).
  • Cognitive Load on Low/Low target — High/Low starters report lower load (d = +0.70, pperm = .033). KW omnibus marginal (p = .062).
  • Position effect on Cognitive Load (Low/Low) — Spearman ρ = −0.24 (p = .024): cognitive load decreases monotonically with later position.
